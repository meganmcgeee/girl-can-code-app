How to contribute to this project.

Here is a brief instructional on how you can contribute to this project to showcase your coding interest/ability and to support the cause of gender equality in the tech sector at the same time!
ALL SKILL LEVELS are welcome to contribute. We seek to make this a place to learn and make mistakes while aiming to give back at the same time. It can be intimidating to get started as a coder and especialyintimidating if you feel like the only female.
