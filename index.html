Girl Can Code is

an information resource and an initiative to create open-source projects that inspire and support aspiring and existing female software engineers. This work is inspired by using Tracy Chou's work, women-in-eng, which is a working list of the perecntage of female software engineers employed full-time residing in the continental United States.

Here you will find resources for the aspiring software engineers and organizations, specifically for aspiring female technologists.

Here's another to fight the good fight.
